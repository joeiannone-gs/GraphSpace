## âš™ï¸ Installation

Clone the repository and start both backend and frontend servers:

```bash
# Clone the repository
git clone https://github.com/joeiannone-gs/GraphSpace
cd graphspace

# Start the compute server
sudo python3.12 -m computeserver.main

# Launch the frontend
cd frontend
sudo npm install
sudo npm run dev
```
